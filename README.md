# Steps

1. Get data from Genbank. (`./scripts/get_data.py`)
2. Running initial alignments (`./scripts/get_alignments.sh`)
3. Trimming Alignments (`./scritps/trim_alignments.R`)
4. Create Supermatrix (`./scripts/supermat.R`)
5. Trees:
   - ML tres (see `./scripts/README.md` for more info)
   - Bayesian trees in BEAST (see `./scripts/README.md` for more info)
